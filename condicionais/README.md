# Condicional "if"

O `if` é uma estrutura condicional usada para executar um bloco de código somente se uma condição for verdadeira.

**`if`** - testa uma condição

```python
if condição:
    # código executado se a condição for verdadeira
```

**`elif`** - testa outra condição se o if for falso

```python
elif outra_condição:
    # código executado se essa condição for verdadeira
```

**`else`** - executa se nenhuma condição anterior for verdadeira

```python
else:
    # código executado se todas as condições anteriores forem falsas
```

Vimos que usamos `if`, `elif` e `else` para tomar decisões. Agora, com operadores lógicos, podemos combinar várias condições em uma só.

- **and** - todas as condições devem ser verdadeiras
- **or** - pelo menos uma condição deve ser verdadeira
- **not** - inverte o valor lógico
