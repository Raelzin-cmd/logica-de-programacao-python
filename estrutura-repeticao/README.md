# For

O for é uma estrutura de repetição usada para executar um bloco de código várias vezes, percorrendo elementos de uma sequência (como listas, strings, tuplas ou intervalos).
Em resumo, o for serve para automatizar repetições sem precisar escrever várias linhas iguais.

Exemplo:

```python
# Usando range para repetir 10 vezes

for num in range(10):   # for percorre pela lista de 10 posições
    print(num)     # cada vez que o for passar, ele vai printar uma posição
```
