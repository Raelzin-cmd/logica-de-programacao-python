# While

O comando while é uma estrutura de repetição enquanto uma condição for verdadeira "Enquanto algo for verdade, continue fazendo isso".

Exemplo prático em [while.py](../while/while.py)

As vezes podemos fazer um loop infinito por uma implementação fora de lógica, um while que se repetirá infinitas vezes caso não seja interrompido. Por isso, usamos o break para interromper o loop.

Exemplo prático em [while-break.py](../while/while-break.py)
