# Match - Case

O `match case` no python é uma forma mais organizada de lidar com várias condições, parecido com o `switch` de outras linguagens.

`Match case` é uma estrutura de controle de fluxo. Serve para verificar o valor de uma variável e executar um bloco de código dependendo do caso e torna o código mais limpo e legível do que usar muitos `if/elif`.
